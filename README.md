## How to Use

1. Enter the full URL of the website you want to download (include http:// or https://)
2. Click the "Download Code" button
3. View the code preview in the application
4. Click "Save Code" to download the full HTML as a file

## Installation

No installation required! Simply:

1. Download the `index.html` file
2. Open it in any modern web browser
3. Start downloading website code

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Limitations

Due to browser security policies (CORS), some websites may not be accessible directly. The tool uses a proxy API to mitigate this, but some sites may still block access.

## Technical Details

This project is built with:
- HTML5
- CSS3 (with 3D transformations and animations)
- Vanilla JavaScript
- [AllOrigins](https://allorigins.win/) CORS proxy API
